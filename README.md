# LegendaryPath(轻量RPG)
## 1.怪物生成的开启
命令: /lp <switch> <world>  
权限: lp.admin  
-- switch选项: open打开 close关闭  
-- world选项: 世界名称,兼容多世界-该插件的世界必须为主世界类型  
## 2.鉴定系统
命令:/identify  
权限:lp.identify  
-- 执行命令会打开鉴定系统GUI  
## 3.淬炼系统
命令:/quench  
权限:lp.quench  
-- 执行命令会打开淬炼系统GUI  
## 4.掉落物
- 未鉴定装备  
- 鉴定书  
- 淬炼石
- 灵魂瓶
